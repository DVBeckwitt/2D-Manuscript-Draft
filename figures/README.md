# Figure assets

Figures are grouped by manuscript section and topic. Filenames are intended to be descriptive of their usage.

## intro
- `reciprocal_single_crystal.png`: Intro figure panel for single-crystal reciprocal space.
- `reciprocal_3d_powder.png`: Intro figure panel for 3D powder reciprocal space.
- `reciprocal_2d_powder.png`: Intro figure panel for 2D powder reciprocal space.
- `realspace_2d_powder.png`: Intro figure panel showing oriented 2D powders.
- `detector_3d_powder_hbn.png`: Intro panel showing a 3D powder detector image (hBN).
- `detector_2d_powder_example.png`: Intro panel showing a 2D powder detector image example.
- `realspace_2d_powder_alt.png`: Alternate 2D real-space schematic (currently unused).

## geometry
- `system_geometry.tex`: Lab-frame WAXS geometry diagram.
- `sample_geometry_rotation.tex`: Sample rotation schematic.
- `sample_geometry_alignment.tex`: Sample alignment schematic with tilt.
- `sample_geometry_no_rotation.tex`: Sample geometry without rotation (currently unused).

## mosaic
- `mosaic_schematic.png`: Mosaicity schematic.
- `mosaic_bragg_inplane.tex`: In-plane Bragg-circle schematic.
- `mosaic_bragg_specular.tex`: Specular Bragg-circle schematic.
- `inplane_detector_band.png`: Detector-space in-plane band example.
- `specular_detector_cap.png`: Detector-space specular cap example.
- `pbi2_waxs.png`: Example PbI2 WAXS pattern.

## theory
- `ewald_single_crystal.tex`: Ewald sphere diagram for a single crystal (currently unused).
- `pseudo_voigt_arc.tex`: Pseudo-Voigt kernel plot (currently unused).
- `mosaic_bragg_sphere_plane.tex`: 3D Bragg-sphere plane illustration (currently unused).
- `mosaic_bragg_sphere_specular.tex`: 3D Bragg-sphere specular illustration (currently unused).
