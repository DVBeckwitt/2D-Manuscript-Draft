\section{Results: diffuse scattering in PbI\texorpdfstring{$_2$}{2}}
\label{sec:results_pbi2}

PbI$_2$ extends the ordered-film analysis to a case where the detector image is not described by Bragg peaks from a single ordered stacking sequence alone. The same calibrated detector geometry, beam phase space, two-dimensional powder construction, and mosaic treatment are still used. The additional physics is stacking disorder: local PbI$_2$ layers remain crystalline enough to define in-plane reciprocal-space radii, but the registry between adjacent layers is not perfectly periodic. That loss of stacking coherence redistributes intensity along $Q_z$ and produces diffuse features that must be modeled rather than subtracted as background.

In an ideal layered crystal, the scattering from successive layers has a fixed phase relation. For a reflection with momentum transfer $\mathbf Q=(\mathbf Q_R,Q_z)$, the intensity contains an interference factor from all layer pairs. If the separation and in-plane registry of every adjacent layer are perfectly repeated, that interference factor is sharply peaked at the ordered Bragg values of $Q_z$. A stacking fault changes the registry of one layer relative to the next. The in-plane order within each layer can remain largely intact, so the same $Q_R$ families are still present, but the phase relation along the stacking direction is interrupted. The result is a combination of residual ordered Bragg intensity and diffuse intensity spread along $Q_z$.

For a two-dimensional powder film, random in-plane rotation turns each in-plane reflection family into a reciprocal-space cylinder at fixed $Q_R$. Stacking disorder then modulates the intensity along that cylinder. In detector space, the measured diffuse features are therefore not arbitrary streaks; they are the detector projection of fault-broadened intensity on these $Q_R$ cylinders after the same beam bandwidth, divergence, mosaicity, detector tilt, and integration effects used for the ordered films are applied.

The stacking-fault model should be introduced as a real-space correlation model rather than as an empirical detector correction. A useful representation is a stack of PbI$_2$ layers in which each layer-to-layer step can follow the ideal registry or one of the allowed faulted registries. The fault probabilities determine the probability that two layers separated by $n$ spacings remain in the same phase relation. Those pair correlations determine the $Q_z$ dependence of the diffuse intensity. In the limit of zero fault probability, the model returns the ordered Bragg sequence. As the fault probability increases, the ordered peaks lose coherent weight and the diffuse component between or around them grows.

The PbI$_2$ comparison should therefore contain three calculated components: an ordered-baseline calculation, a stacking-disorder calculation, and the measured detector-space or projected $Q_z$ profile to which both are compared. The ordered baseline shows which features would be expected from a perfectly periodic stack. The disorder calculation tests whether the chosen fault probabilities and layer-pair correlations reproduce the diffuse intensity without damaging the detector geometry and mosaicity already established by the ordered-film analysis.

This section is intentionally assembled as a figure-driven scaffold. The final version should keep the PbI$_2$ result subordinate to the ordered-film validation unless the diffuse-scattering figures are complete enough to support an independent conclusion. The scientific role of PbI$_2$ is to demonstrate that the same detector-space framework can be extended from ordered Bragg line-shape refinement to disorder-sensitive diffuse scattering when the stacking model is expressed in reciprocal-space terms that map onto the measured detector image.
