\section{Results: diffuse scattering in PbI\texorpdfstring{$_2$}{2}}

We next apply the same calibrated forward framework to disordered CVD-grown PbI$_2$.
Unlike the ordered films, the PbI$_2$ data are dominated by diffuse intensity along the
Bragg rods and between the sharp reflections in $Q_z$, which indicates that the average
ordered structure is not sufficient on its own. The relevant question here is not simply
whether the Bragg positions are correct, but whether the model can reproduce how the
intensity is redistributed when the stacking sequence contains faults.

The physical picture is summarized in Fig.~\ref{fig:reciprocal_cylinders} and expanded in
the supporting information. In the ordered limit, each layer stack produces sharp Bragg
features. Stacking faults interrupt the perfect periodicity and transfer intensity from those
ideal peaks into diffuse rods and background between the peaks. We therefore keep the ordered
layer structure fixed and introduce a Hendricks--Teller type disorder parameterization to
describe the faulted stacking sequence. This keeps the average structure and the disorder
problem separate, which is important because the two effects are otherwise strongly correlated.

Figure~\ref{fig:pbi2_disorder_comp} compares the measured PbI$_2$ image with the
ordered-only simulation and with the stacking-fault model. The key point to emphasize is that
the ordered model can reproduce the basic reciprocal-lattice locations but not the observed
diffuse redistribution of intensity, whereas the faulted model captures both the diffuse rods
and the relative intensity between neighboring peaks. Any residual mismatch should be discussed
as a limitation of the disorder parameterization rather than as a failure of the detector
calibration, because the same geometry already succeeded on the ordered films.

The supporting information contains the full derivation of the stacking-fault formalism and the
polytype background, so the main text only needs this compact physical summary.

Overall, the PbI$_2$ result demonstrates the main extension of the paper: once the ordered
film case is established, the same forward model can be expanded to include stacking faults and
used to interpret diffuse scattering quantitatively rather than treating it as an unmodelled
background.
